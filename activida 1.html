activiad 2 semana 3

html:es el lenguaje que se emplea para el desarrollo de páginas de internet,tiene etiquetas para imágenes, hipervínculos que nos permiten dirigirnos a otras páginas, saltos de línea, listas, tablas, etc.

body:El elemento body es el contenedor para todos los aspectos representables de un documento. Este es el lugar donde los autores arrojan todos los elementos y contenidos que sus visitantes alcanzarán.

a: Es una etiqueta que nos ayuda a poder crear un enlace a una página web. El atributo principal de la etiqueta HTML es href, donde pondremos el enlace al que queremos conectar.

img:representa una imagen en el documento,sí la persona elige no mostrar la imagen, o sí el navegador es incapaz de mostrarla porque no es válida o soportada.

br: produce un salto de línea en el texto (retorno de carro). Es útil para escribir un poema o una dirección, donde la división de las líneas es significante.

strong: El elemento strong es el apropiado para marcar con especial énfasis las partes más importantes de un texto.

h1:es utilizado, de manera habitual, para identificar la cabecera más importante en una página web-

head:provee información general (metadatos) acerca del documento, incluyendo su título y enlaces a scripts y hojas de estilos.

p: El elemento p (párrafo) es el apropiado para distribuir el texto en párrafos.

li: El elemento li representa a un ítem en una lista, ya sea ordenada (elemento ol ) o no ordenada (elemento ul ).
